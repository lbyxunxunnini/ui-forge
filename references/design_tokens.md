# 设计Token

## 颜色系统

- Primary: #667eea
- Secondary: #764ba2
- Neutral: #f5f5f7 - #0f172a
- Success: #22c55e
- Warning: #f59e0b
- Error: #ef4444

### 颜色使用规范

- 主色调用于按钮、链接、激活状态
- 中性色用于背景、文字、边框
- 成功色用于成功提示
- 警告色用于警告提示
- 错误色用于错误提示

## 字体系统

- Primary: -apple-system, BlinkMacSystemFont, 'SF Pro Display'
- H1: 24px, Bold
- H2: 20px, Semibold
- Body: 16px, Regular
- Caption: 14px, Regular

### 字体使用规范

- H1用于页面标题
- H2用于卡片标题
- Body用于正文内容
- Caption用于辅助文字

## 间距系统

- XS: 4px
- SM: 8px
- MD: 16px
- LG: 24px
- XL: 32px
- XXL: 48px

### 间距使用规范

- XS用于图标与文字间距
- SM用于列表项间距
- MD用于组件内边距
- LG用于卡片内边距
- XL用于页面内边距
- XXL用于大区块间距

## 圆角

- Small: 8px
- Medium: 14px
- Large: 20px
- XLarge: 24px

### 圆角使用规范

- Small用于小按钮、标签
- Medium用于输入框、按钮
- Large用于卡片
- XLarge用于弹窗、模态框

## 阴影

- Small: 0 2px 8px rgba(0, 0, 0, 0.06)
- Medium: 0 4px 16px rgba(0, 0, 0, 0.08)
- Large: 0 8px 32px rgba(0, 0, 0, 0.1)

### 阴影使用规范

- Small用于按钮、标签
- Medium用于卡片、下拉菜单
- Large用于弹窗、模态框
