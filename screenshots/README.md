# 截图目录

本目录用于存放 Demo 的截图预览。

## 如何创建截图

1. 在浏览器中打开 Demo 页面
2. 使用系统截图工具：
   - macOS: `Cmd + Shift + 4` 或 `Cmd + Shift + 3`
   - Windows: `Win + Shift + S` 或 `PrtSc`
3. 保存截图到本目录

## 建议的截图

| 文件名 | 说明 |
|--------|------|
| `login-modern.png` | 登录页 - 现代风格 |
| `login-minimalist.png` | 登录页 - 简约风格 |
| `login-business.png` | 登录页 - 商务风格 |
| `login-tech.png` | 登录页 - 科技风格 |
| `login-playful.png` | 登录页 - 活泼风格 |
| `home-modern.png` | 首页 - 现代风格 |
| `home-tech.png` | 首页 - 科技风格 |

## 截图规范

- 尺寸：建议 375x812px（iPhone X）或 1440x900px（桌面）
- 格式：PNG
- 背景：保持 Demo 原始背景
